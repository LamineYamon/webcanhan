HƯỚNG DẪN SỬ DỤNG THƯ MỤC IMAGES
=================================

Thư mục này chứa các hình ảnh cho website cá nhân.

Các file hình ảnh cần thiết:
---------------------------

1. profile.jpg (300x300px) - Ảnh profile cho trang chủ
2. about-me.jpg (400x400px) - Ảnh cho trang giới thiệu  
3. project1.jpg (400x250px) - Ảnh dự án 1
4. project2.jpg (400x250px) - Ảnh dự án 2
5. project3.jpg (400x250px) - Ảnh dự án 3
6. project4.jpg (400x250px) - Ảnh dự án 4
7. default-project.jpg (400x250px) - Ảnh mặc định khi không có ảnh dự án

Định dạng được khuyến nghị:
--------------------------
- JPG hoặc PNG
- Kích thước theo khuyến nghị ở trên
- Tối ưu hóa để web (dưới 500KB mỗi file)

Cách thêm hình ảnh:
------------------
1. Copy các file hình ảnh vào thư mục này
2. Đặt tên file đúng theo danh sách ở trên
3. Restart server để áp dụng thay đổi

Ghi chú:
--------
- Website sẽ vẫn hoạt động bình thường ngay cả khi thiếu hình ảnh
- Có thể sử dụng hình ảnh placeholder từ internet trong quá trình phát triển
